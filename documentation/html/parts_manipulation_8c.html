This function removes all the parts with a specific class.

This function iterates through the parts list checking if the actual part class is the same class that the user wants to remove.Then it checks if there is any part after the actual part and sets the next part previous part to the actual part previous part, then it checks if there is any previous part, and if there is it will set the varible next part from the previous part to the actual part next part. After all of that clutter it deletes the actual node and repeates the cycle.
