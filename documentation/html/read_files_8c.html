PARTS* OpenParts ( PARTS *  parts)

This function reads the parts.tsv file and inserts its values in a parts list.

This function firstly checks if it can open the files (checking if the file exists), then it uses the fseek function and jumps the first line, after that cycles through the file reading line by line and storing each line values in a list that is then returned
