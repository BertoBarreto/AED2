This function allows to inset a part into the PARTCOUNTER list, it checks if the list has values in the actual positon, if not then it will add the new part with counter to 1, else if the part that is in the actual position is the same that we are trying to insert then the counter of that part will be increased, if none of those conditions verifies then it will check the next position.
